### Installation

_Below is an example of how you can be installing app._

1. Clone the repo
   ```sh
   git clone https://github.com/PiotrSierant/MyMusicTask.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start project
   ```sh
   npm start
   ```
